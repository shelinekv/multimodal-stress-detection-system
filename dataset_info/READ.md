# Dataset Description

## CK+48 Emotion Dataset

This project uses a facial emotion recognition dataset containing 6 emotion classes,
structured into training and testing sets.

### Emotion Classes:
- Happy
- Sad
- Angry
- Fear
- Surprise
- Neutral

### Dataset Structure:
- train/
- test/

Each class contains facial images used for training and evaluation of machine learning
and deep learning models.

### Usage in Project:
The dataset is used for:
- Facial emotion recognition
- Stress detection modelling
- AI model training and testing

### Note:
The full dataset is not uploaded to this repository due to size constraints.
Please use standard CK+48 dataset sources or request access if required.
